Подробное руководство по WhatWeb

WhatWeb - это инструмент с открытым исходным кодом, который идентифицирует технологии, используемые на веб-сайтах. Он может обнаруживать различные компоненты, включая системы управления контентом (CMS), веб-серверы, языки программирования, библиотеки JavaScript и многое другое.

Что такое WhatWeb?

WhatWeb помогает получить информацию о технологическом стеке веб-сайта. Это полезно для различных целей, включая:

  • Пентестинг: Определение потенциальных уязвимостей на основе используемых технологий.
  • Разведка: Сбор информации о веб-сайте перед проведением дальнейших тестов.
  • Анализ: Понимание технологий, используемых конкурентами или другими веб-сайтами.

Как работает WhatWeb?

WhatWeb работает, отправляя HTTP-запросы к целевому веб-сайту и анализируя ответы. Он использует различные методы для обнаружения технологий, включая:

  • Поиск определенных заголовков HTTP.
  • Анализ HTML-кода на наличие определенных шаблонов.
  • Проверку наличия определенных файлов.
  • Использование регулярных выражений для поиска характерных признаков.

Основные возможности WhatWeb

WhatWeb обладает множеством полезных функций:

  • Обширная база данных: WhatWeb имеет большую базу данных сигнатур для обнаружения множества различных технологий.
  • Низкий уровень шума: Инструмент разработан для минимизации ложных срабатываний.
  • Настраиваемый: WhatWeb можно настроить для обнаружения определенных технологий или изменения поведения сканирования.
  • Поддержка различных форматов вывода: Результаты можно вывести в различных форматах, таких как текстовый, XML и JSON.
  • Автоматизация: WhatWeb можно использовать в скриптах для автоматизации сканирования.

Примеры использования WhatWeb

Базовое сканирование

Этот пример демонстрирует, как выполнить базовое сканирование веб-сайта. 


# Выполнение базового сканирования:
whatweb http://example.com

# Что вы увидите в терминале:
# WhatWeb начнет сканирование http://example.com и отобразит обнаруженные технологии.
# Пример вывода:
# http://example.com [200 OK] Country[UNITED STATES][us], HTTPServer[Apache], IP[192.0.2.1], X-Powered-By[PHP 7.2.34], PHP[7.2.34], ...

Сканирование нескольких сайтов

Этот пример показывает, как сканировать несколько веб-сайтов одновременно. 


# Сканирование нескольких сайтов:
whatweb http://example.com http://example.org http://example.net

# Что вы увидите в терминале:
# WhatWeb просканирует каждый из указанных сайтов и отобразит результаты для каждого.

Вывод в формате JSON

Этот пример демонстрирует, как вывести результаты в формате JSON. 


# Вывод в формате JSON:
whatweb -f json http://example.com

# Что вы увидите в терминале:
# WhatWeb выведет результаты сканирования в формате JSON.
# Пример вывода:
# {
#  "target":"http://example.com",
#  "plugins":{
#   "ip":{
#    "string":"192.0.2.1"
#   },
#   "httpserver":{
#    "string":"Apache"
#   },
#   ...
#  }
# }

Настройка агрессивности сканирования

Этот пример показывает, как настроить уровень агрессивности сканирования. 


# Установка агрессивности на 3 (самый агрессивный):
whatweb -a 3 http://example.com

# Что вы увидите в терминале:
# WhatWeb будет выполнять более агрессивное сканирование, что может занять больше времени и увеличить количество запросов к серверу.

Важные замечания

  • Используйте WhatWeb только на тех системах, которые вам разрешено тестировать. Несанкционированное использование WhatWeb может привести к юридическим последствиям.
  • Будьте осторожны при сканировании веб-серверов. Чрезмерное сканирование или использование высокой агрессивности может привести к блокировке вашего IP-адреса или вызвать проблемы с производительностью сервера.
  • Всегда анализируйте результаты сканирования вручную, чтобы избежать ложных срабатываний.
